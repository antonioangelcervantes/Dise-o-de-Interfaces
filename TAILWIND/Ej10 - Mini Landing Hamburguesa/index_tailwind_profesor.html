Características principales

Esta mini web incluye maquetación responsive, cabecera sticky, menú adaptable a móvil y un pequeño comportamiento con JavaScript.

  • Diseño mobile first
  • Menú hamburguesa en móviles
  • HTML semántico y estructura clara

Sobre este proyecto

Este ejemplo está pensado como práctica para el módulo de Diseño de Interfaces Web, combinando maquetación y JavaScript de forma sencilla y realista.
